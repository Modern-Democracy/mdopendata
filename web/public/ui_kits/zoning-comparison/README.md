# Zoning Comparison — UI kit

Side-by-side viewer for current vs. draft Charlottetown zoning. Changed values in ochre, added rules in green, removed rules in brick. Summary counts at the top; diff-row list underneath.

## Components

- Header (shared with parcel-lookup)
- Comparison summary bar (changed / added / removed / unchanged counts)
- Diff-row (current · label · draft)
- Status markers: `WAS`, `NOW`, `NEW`, `REMOVED`
